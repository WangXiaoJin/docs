## 常用

* #### 禁用插件
    如果插件配置项支持`skip`配置，则用以下方案：
    
    ```xml
    <plugin>
        <groupId>group</groupId>
        <artifactId>artifact</artifactId>
        <configuration>
            <skip>true</skip>
        </configuration>
    </plugin>
    ```
    
    不支持`skip`配置，则用以下方案：
    
    ```xml
    <plugin>
        <groupId>group</groupId>
        <artifactId>artifact</artifactId>
        <executions>
            <execution>
                <phase>none</phase>
            </execution>
        </executions>
    </plugin>
    ```

* #### 显示`final pom`配置

    ```bash
    shell> mvn help:effective-pom
    ```

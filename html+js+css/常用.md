## 常用

* **JS二进制、十进制、十六进制转换**
    
    * 2的8次方 - `Math.pow(2, 8)`
    * 二进制转十进制 - `parseInt("111", 2)`
    * 八进制表示 - `011 == 9`
    * 十六进制表示 - `0x11 == 17`
    * 十进制转二进制 - `new Number("11").toString(2)`
    * 十进制转十六进制 - `new Number("11").toString(16)`
    * 十六进制转二进制 - `new Number("0x11").toString(2)`

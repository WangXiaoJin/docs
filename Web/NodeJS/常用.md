#### npm

* `npm config ls` 显示当前配置。`npm config ls -l`显示当前配置及`默认配置`。

* 配置`prefix`/`cache`参数

  ```bash
  # 全局模块插件存放路径，默认值为安装目录
  npm config set prefix "D:\Program Files\node-v10.15.3"
  
  # 缓存路径，默认值 ~AppData\Roaming\npm-cache
  npm config set cache "D:\Program Files\node-v10.15.3\node_cache"
  ```
